# @image-charts/pcf-all

> Complete Image-Charts PCF Component Suite for Microsoft Power Apps

This meta-package includes all Image-Charts PCF components:

- **[@image-charts/pcf-qrcode](https://www.npmjs.com/package/@image-charts/pcf-qrcode)** - QR Code Generator
- **[@image-charts/pcf-barchart](https://www.npmjs.com/package/@image-charts/pcf-barchart)** - Bar Chart Generator (vertical & horizontal)
- **[@image-charts/pcf-piechart](https://www.npmjs.com/package/@image-charts/pcf-piechart)** - Pie Chart Generator
- **[@image-charts/pcf-linechart](https://www.npmjs.com/package/@image-charts/pcf-linechart)** - Line Chart Generator
- **[@image-charts/pcf-doughnut](https://www.npmjs.com/package/@image-charts/pcf-doughnut)** - Doughnut Chart Generator
- **[@image-charts/pcf-generic](https://www.npmjs.com/package/@image-charts/pcf-generic)** - Generic Chart Generator (supports all chart types)

## Installation

```bash
npm install @image-charts/pcf-all
```

## Usage

After installation, you'll find the combined Power Apps solution at:
- `node_modules/@image-charts/pcf-all/solution/ImageChartsAll.zip`

Import this solution into your Power Apps environment to get all components at once.
